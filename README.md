## 概要
* Spresenseを使ったカメラの組み込みコードです
* カメラのストリーミング画像にエフェクトを加え、SDカードに保存できます
* エフェクト処理やディスプレイ描画処理を変更して、新しいカメラのアイデアを試せます

## 使用機材
* Spresense メインボード
* Spresense 拡張ボード 
* Spresense カメラボード
* ILI9341 液晶ディスプレイ
* SDカード
* スイッチ×5

## 表示画像
* DALL-Eにて作成
